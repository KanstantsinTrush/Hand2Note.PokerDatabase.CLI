# Hand2Note Poker Database CLI

CLI приложение для работы с базой данных покерных раздач из истории рук PokerStars.

## Запуск проекта

### Требования
- .NET 9.0 SDK

### Сборка проекта
```bash
dotnet build
```

### Запуск приложения
```bash
dotnet run -- <путь_к_директории_с_историями>
```

Или после сборки:
```bash
./bin/Debug/net9.0/Hand2Note.PokerDatabase.CLI <путь_к_директории_с_историями>
```

**Пример:**
```bash
dotnet run -- ./test-data
```

Приложение загрузит все файлы с историями рук из указанной директории (включая поддиректории), распарсит их и запустит интерактивный CLI для работы с базой данных.

## Доступные команды

После запуска приложения вы можете использовать следующие команды в интерактивном режиме:

### ShowStats (или Stats)
Показывает общую статистику по базе данных: общее количество раздач и уникальных игроков.

**Использование:**
```
ShowStats
```
или
```
Stats
```

### ShowPlayerHands (или PlayerHands)
Показывает статистику по конкретному игроку: общее количество раздач и последние 10 раздач с информацией о картах, стеке и валюте.

**Использование:**
```
ShowPlayerHands --PlayerName "ИмяИгрока"
```
или
```
ShowPlayerHands -p "ИмяИгрока"
```
или
```
PlayerHands -p "ИмяИгрока"
```

**Параметры:**
- `--PlayerName` или `-p` - имя игрока (обязательный параметр)

### DeleteHand
Удаляет раздачу с указанным номером из базы данных.

**Использование:**
```
DeleteHand --HandNumber 123456789
```
или
```
DeleteHand -n 123456789
```

**Параметры:**
- `--HandNumber` или `-n` - номер раздачи (обязательный параметр)

### ShowDeletedHands (или DeletedHands)
Показывает список номеров удаленных раздач.

**Использование:**
```
ShowDeletedHands
```
или
```
DeletedHands
```

## Выход из приложения

Для выхода из интерактивного режима введите:
```
exit
```
или
```
quit
```

## Навигация по истории команд

В интерактивном режиме можно использовать стрелки вверх/вниз для навигации по истории выполненных команд (последние 5 команд).

## Примеры использования

1. Запуск приложения и просмотр статистики:
```bash
dotnet run -- ./test-data
> ShowStats
```

2. Просмотр раздач конкретного игрока:
```bash
> ShowPlayerHands -p "PlayerName"
```

3. Удаление раздачи:
```bash
> DeleteHand -n 93404843251
```

4. Просмотр удаленных раздач:
```bash
> ShowDeletedHands
```
